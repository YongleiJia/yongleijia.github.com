git 上传数据到github

发表于4月 17 2014|更新于1月 23 2021|hexo
|浏览量:|评论数:

  1. 第一步:将代码从代码仓库中下载下来

    git clone https://github.com/snailgoer\/snailgoers.github.com.git

  2. 第二步:添加所有文件

    git add .

  3. 第三步:

    git commit -m ‘The first add’

  4. 第四步:

    git remote rm origin

  5. 第五步:

    git remote add origin https://github.com/snailgoers/snailgoers.github.com.git
    git push origin master
